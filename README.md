# 周波数チューナー

## 概要

周波数を自由に入力し、Web Audio APIで音を生成・再生する個人用の試作Webアプリです。スマートフォンの縦画面とiPhone Safariでの利用を優先し、ホーム画面へ追加した場合にも使いやすい画面構成にしています。

## 公開URL

https://heat-mix.github.io/frequency-tuner/

## 主な機能

- 単音モード：周波数、波形、音量、左右バランスの設定とWAV書き出し
- 左右差モード：左右周波数と差分Hzの連動、左右別の音量・バランス設定
- プリセットモード：定番周波数の選択と単音・左右差モードへの反映
- ビートメトロノーム：Hz入力、BPM換算、クリック音再生
- ノイズモード：ホワイト、ピンク、ブラウン、ブルー、バイオレットノイズ
- 音声リセット：AudioContextを閉じて音声エンジンを再作成
- iPhoneのホーム画面追加に対応するアイコンとWeb App Manifest

## GitHub Pagesでの公開

このフォルダ内のファイルをリポジトリ直下へ配置します。GitHubの `Settings` → `Pages` で、公開元を `Deploy from a branch`、ブランチを `main`、フォルダを `/(root)` に設定すると公開できます。

すべてのアセットは相対パスで参照しているため、追加のビルド処理やNetlify専用設定は不要です。
